Phyton

O que é Phyton?

é uma das linguagens de programação mais populares e versáteis do mundo. Ele é conhecido pela sua sintaxe simples e limpa, muito parecida com a linguagem humana, o que o torna ideal tanto para iniciantes quanto para programadores experientes.

Qual a diferença entre uma lista e uma tupla em Python?

A lista é mutável, ou seja, seus elementos podem ser alterados após a criação. Já a tupla é imutável, não permitindo modificações depois de criada.

Para que serve a função print() em Python?

A função print() é utilizada para exibir informações na tela.

Como declarar uma variável em Python?

Em Python, não é necessário informar o tipo da variável. Basta atribuir um valor

Como criar uma estrutura de repetição em Python?

A estrutura for é usada para repetir ações várias vezes.

TypeScript

O que é TypeScript?

TypeScript é uma linguagem de programação desenvolvida pela Microsoft que adiciona tipagem estática ao JavaScript. Ela é compilada para JavaScript e ajuda a criar aplicações mais seguras e fáceis de manter.

Como declarar uma variável do tipo string em TypeScript?

Em TypeScript, é possível definir explicitamente o tipo da variável

O que é uma interface em TypeScript?

Uma interface define a estrutura que um objeto deve seguir

Qual a diferença entre any e unknown?

O tipo any desativa a verificação de tipos, permitindo qualquer operação. Já unknown é mais seguro, pois exige que o tipo seja verificado antes de ser utilizado.

Como criar uma função com tipos definidos em TypeScript?

Os tipos dos parâmetros e do retorno podem ser especificados diretamente na função.

JavaScript

O que é JavaScript

JavaScript é uma linguagem de programação amplamente utilizada para criar páginas web interativas. Ela permite adicionar funcionalidades como animações, validação de formulários e comunicação com servidores.

Qual a diferença entre let, const e var?

let: permite alterar o valor da variável. const: cria uma constante cujo valor não pode ser reatribuído. var: forma antiga de declarar variáveis, com regras de escopo diferentes..

Como criar uma função em JavaScript?

Uma função pode ser criada usando a palavra-chave function.

O que é um array em JavaScript?

Um array é uma estrutura que permite armazenar vários valores em uma única variável.

O que é uma Promise em JavaScript?

Uma Promise representa o resultado futuro de uma operação assíncrona, podendo estar nos estados de pendente, resolvida ou rejeitada.

Java

O que é Java?

Java é uma linguagem de programação orientada a objetos, multiplataforma e amplamente utilizada no desenvolvimento de aplicações web, desktop, mobile e corporativas. Seu lema é "Write Once, Run Anywhere" (Escreva uma vez, execute em qualquer lugar).

Como declarar uma variável em Java?

Em Java, é necessário informar o tipo da variável no momento da declaração.

O que é uma classe em Java?

Uma classe é um modelo que define atributos e métodos para criar objetos.

Qual a diferença entre == e .equals() em Java?

O operador == compara referências de objetos, enquanto o método .equals() compara o conteúdo dos objetos.

Como funciona um laço for em Java?

O laço for é usado para repetir um bloco de código um número determinado de vezes.

Go (Golang)

O que é Go (Golang)?

Go, também conhecido como Golang, é uma linguagem de programação de código aberto criada pelo Google. Ela foi projetada para ser simples, rápida e eficiente, sendo muito utilizada no desenvolvimento de APIs, microsserviços, sistemas distribuídos e aplicações em nuvem.

Como declarar variáveis em Go?

Em Go, as variáveis podem ser declaradas usando a palavra-chave var ou a sintaxe curta

O que é uma goroutine em Go?

Uma goroutine é uma função executada de forma concorrente. Ela permite que múltiplas tarefas sejam executadas simultaneamente com baixo custo de recursos.

Como criar uma função em Go?

As funções são definidas com a palavra-chave func.

Para que servem os canais (channels) em Go?

Os canais são usados para comunicação segura entre goroutines, permitindo a troca de dados sem a necessidade de mecanismos complexos de sincronização.

C#

O que é C#?

C# (pronuncia-se "C Sharp") é uma linguagem de programação orientada a objetos desenvolvida pela Microsoft. Ela é amplamente utilizada para criar aplicações desktop, web, mobile, jogos e serviços usando a plataforma .NET.

Como declarar uma variável em C#?

Em C#, é necessário especificar o tipo da variável ou usar a palavra-chave var quando o tipo puder ser inferido.

O que é uma classe em C#?

Uma classe é um modelo que define propriedades e métodos para criar objetos.

Como criar um método em C#?

Métodos são blocos de código que executam uma tarefa específica e podem retornar valores.

O que é async e await em C#?

As palavras-chave async e await são usadas para programação assíncrona, permitindo executar tarefas sem bloquear a aplicação.

C++

O que é a linguagem C?

C é uma linguagem de programação de propósito geral criada por Dennis Ritchie. Ela é amplamente utilizada no desenvolvimento de sistemas operacionais, drivers, sistemas embarcados e aplicações de alto desempenho.

Como declarar uma variável em C?

Em C, é necessário informar o tipo da variável durante a declaração.

Como exibir uma mensagem na tela em C?

A função printf() é utilizada para exibir informações no console.

O que é um ponteiro em C?

Um ponteiro é uma variável que armazena o endereço de memória de outra variável.

Como funciona um laço for em C?

O laço for permite repetir um bloco de código várias vezes.
